# 💱 Conversor de Moedas

Um conversor de moedas moderno desenvolvido com HTML, CSS e JavaScript, consumindo cotações em tempo real através de API.

## 🚀 Demonstração


https://willbarreto23.github.io/conversor-moedas/


## ✨ Funcionalidades

* Conversão entre:

  * 🇧🇷 Real Brasileiro (BRL)
  * 🇺🇸 Dólar Americano (USD)
  * 🇪🇺 Euro (EUR)
  * 🇬🇧 Libra Esterlina (GBP)
  * ₿ Bitcoin (BTC)

* Conversão em tempo real utilizando API de cotações.

* Interface moderna e responsiva.

* Borda animada com efeito neon.

* Atualização dinâmica das bandeiras e moedas.

* Compatível com dispositivos móveis, tablets e desktop.
<br>

## 🛠️ Tecnologias Utilizadas

### HTML5

* Estrutura semântica da aplicação.
* Formulários e elementos de interface.

### CSS3

* Flexbox
* Responsividade com Media Queries
* Animações com Keyframes
* Efeitos Neon
* Gradientes Animados

### JavaScript (ES6+)

* Manipulação do DOM
* Eventos
* Async/Await
* Fetch API
* Formatação monetária com Intl.NumberFormat
